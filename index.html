CMS error code tool

This is an app I created using python with tkinter.

The app is a simple GUI to view the error codes for A CMS horizontal expander machine. The codes were available to me as an operator only in the form of a pdf. I parsed through the pdf to get the instructions for each code. I then complied those codes into a json to use as a searchable database. The app is a simple GUI with search and logging features. Mixamo to Godot Script

This is a script I created to automate adding root motion to Mixamo animation files in Blender 2.9

I created this script as a time-saver for rigging animation armatures in Blender. It condenses several animation files into one armature. It also formats those animations with the singular armature for export into the Godot game engine.
